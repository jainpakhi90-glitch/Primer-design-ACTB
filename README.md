# Primer-design-ACTB
Title :- Primer Design for Human ACTB Gene
Objective :- To design gene- specific PCR primers for ACTB gene ensuring high specificity.
Gene selected:- ACTB (Beta-actin), Homo sapiens
Methodology:-
•	Gene sequence retrieved from NCBI
•	Primers designed using Primer-BLAST
•	Parameters: Tm 58-60 degree Celsius , GC 40-60%,product size 100-200 bp
•	Specificity checked against human genome
Results:-
•	Forward Primer (5’-3’)- CCCTGAGCTTCGAGCAGAG
•	Reverse Primer ( 5’-3’)- CGGTAACCCCTAGTCAGGC
•	GC Content :- 60%
•	Melting Temperature :- 60.11 degree Celsius
•	Expected PCR product size :- 190 bp
  
Summary:-
•	Specificity Check was performed using Primer-BLAST against the Homo sapiens genome. 
•	No significant off-target binding was observed.
